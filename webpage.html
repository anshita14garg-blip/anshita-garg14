1. Headings

H1 — <h1> is typically the page title.
H2 — <h2> is for major sections.
H3 — <h3> is for subsections.

Italic text — use <em>

3. Special Characters

Some common HTML entities and characters:

  • &amp; renders as: &
  • &lt; renders as: <
  • &gt; renders as: >
  • &copy; renders as: ©
  • &nbsp; is a non-breaking space (prevents line breaks)
  • Numeric entity example: © renders as ©

4. Ordered List

An ordered list (<ol>) is used when order matters:

  1. Introduction to the course
  2. Core modules
    1. Module A
    2. Module B
  3. Assessment & grading

5. Unordered List

An unordered list (<ul>) is used for items where order does not matter:

  • Resources required
  • Recommended reading
    • Book 1
    • Book 2
  • Office hours

6. Description List

Use a description list (<dl>) for term/definition pairs:

Lecture
A scheduled class where an instructor presents material.
Lab
Hands-on practical session to reinforce concepts.
Assignment
Work submitted by students to assess learning.

7. Nested Lists

Lists can be nested to represent hierarchy:

  • Semester 1
    1. Week 1: Introduction
    2. Week 2: Fundamentals
      • Topic A
      • Topic B
  • Semester 2

8. Mixed Lists

You can mix ordered and unordered lists to convey different levels of importance:

  1. Project Phases
    • Phase 1 — Planning
    • Phase 2 — Implementation
  2. Evaluation
    1. Continuous assessment
    2. Final exam
